Info about the content and directory structure of the Abdra Commander project

File/folderDescription
/srcMain source folder.
/src-utilSource of some utility/test application, not part of the Abdra Commander application.
/app-folder/Contains the off-line files and static files/folders required to run Abdra Commander.
/LanguagesThe language files (translations and localized settings) of Abdra Commander.
/libExternal Java libraries required and used by Abdra Commander.
Also contains static resources which are intentionally not placed (or cannot be placed) inside jars.
/docs/Contains documentation related to the development of the project and to the project development.
/auto-generatedDoc and info files not written manually but are generated by utilities or by the Ant build process.
/issue-tracker.htmlIssue and idea tracker static html file.
/release/Folder to deal with release builds.
/release-resourcesResources required for or related to building releases.
/AbdracmdThis is where a release build is assembled.
/resources/Some additional resources used in the project development.
/secureSecurity resources (RSA public and private keys) used for encrypting and decrypting data (e.g. registration info) for Abdra Commander.
/Abdracmd-Dev.launchEclipse launch configuration to run Abdra Commander in development mode.
/Abdracmd.launchEclipse launch configuration to run Abdra Commander in normal mode.
/build.xmlAnt buildfile for various tasks including building a release and creating a project backup.
/content-info.htmlThis document.
